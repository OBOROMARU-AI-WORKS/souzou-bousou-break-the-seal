# 創造暴走 -Break the Seal-

Official-style teaser website for the fictional original anime concept project by 朧丸 / OBOROMARU AI WORKS.

## Files

- `index.html` - one-page teaser site structure and content
- `style.css` - responsive black/crimson tech-gothic styling
- `script.js` - mobile menu and reveal animation
- `assets/` - production images and audio

## Preview

The site is static and does not need a build step. For the most accurate preview, run a local server from this folder and open `http://localhost:5500`.

```bash
python -m http.server 5500
```

Opening `index.html` directly can work for most layout checks, but embedded YouTube playback may require a local server or deployed URL.
